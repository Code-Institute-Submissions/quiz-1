Instructions

Quiz time is a simple multiple choice quiz. When the quiz page is loaded the question will be displayed at the top and the 4 choices will be displayed below. When you think you have the correct answer click the corresponding button and if it's correct a green tick will pop up, and a red x will be displayed if it's incorrect. After answering the question click on the next button to go to the next question, you have to answer the current question before the next one can be displayed. There is a time limit, depending on how many questions you choose (1 minute per 5 questions) and the game will end when the timer runs out.
